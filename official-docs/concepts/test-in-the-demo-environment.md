For testing purposes, Kalshi offers a demo environment with mock funds. You can access the Demo environment at https://demo.kalshi.co/. For safety, credentials are not shared between this environment and production.
To set up a Kalshi Demo account, follow this step-by-step tutorial.
Demo’s recommended Trade API root is https://external-api.demo.kalshi.co/trade-api/v2.
Surface	Recommended demo endpoint	Also supported
REST Trade API	https://external-api.demo.kalshi.co/trade-api/v2	https://demo-api.kalshi.co/trade-api/v2
WebSocket API	wss://external-api-ws.demo.kalshi.co/trade-api/ws/v2	wss://demo-api.kalshi.co/trade-api/ws/v2
For the full production and demo endpoint list, see API Environments and Endpoints.
